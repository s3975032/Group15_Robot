Tools and Technologies

In terms of software, the project needs to build a mobile application on a smartphone and manipulate the robot’s motion and behavior in specific environments. Firstly, programming languages such as Python, Java or C++ can be accomplished with the help of free software development platforms like IDEs or Android studio which are relevant to the appropriate chosen programming language. Furthermore, the requirement for 3D printing is also important for generating a three-dimensional prototype and manufacturing components of the robot.

According to hardware, the hardware components are needed for the modules including motors, sensors, microprocessor like Raspberry Pi or microcontroller like Arduino and Bluetooth and Wi-Fi modules for connecting to the mobile app. More specifically,

  • Robot operating system (ROS): it is an adjustable framework that helps users to write robot software, create robot applications, and manage hardware interfaces with better connection between their components.
  • Computer vision: with the help of computer vision in libraries like OpenCV, the alarm clock robot can adapt to specific environments.
  • Simulators: my team does not need to create physical hardware, so simulators can help us test our robot’s algorithms about their behavior and communication in realistic environments like Gazebo.
  • Machine learning: adding machine learning algorithms can improve the intelligence about decision-making, obstacle detection of robots.
  • Cloud services: integrating cloud services in some platforms like AWS or Google Cloud can enhance the ability to store and process numerous data and use further processing power.
  • Others:Natural language processing (NLP), Mobile App Development.

Those are just some of the project requirements and going into practice will require more tools and technologies. However, it is undeniable that those hardware components depend on the interest, complexity, and finance of the group for choosing specific models and brands. Besides, almost our members just have fundamental experience about some programming languages like Python and Java and approach a few hardware concepts which we will be covering in the upcoming semester.
